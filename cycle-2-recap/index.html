Fuse 2 · regression guards

Playwright in ephemeral, per-branch environments

Prove it before it lands

  • Playwright @react specs run against per-branch / per-PR environments — a feature branch is verified in a disposable env before it reaches main.
  • Umbrella-branch model keeps the full e2e run gating merge to main.
  • Regressions caught pre-merge, not in end-of-cycle QA.

In-house visual-parity gate

  • Built, not bought — frontend-owned tradify-visual-parity.
  • Angular-vs-React capture · pixelmatch diff · masked dynamic zones · ~0.1% threshold.
  • Accepted-discrepancy rules encoded so the gate flags real divergence only.

Fuse 3 · design loop

Productive feedback loops with design

A loop, not a hand-off

Fuse 4 · shared-api

Shared-API usage & contracts

From a couple of reads → the write backbone
