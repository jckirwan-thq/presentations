Two parallel token universes

The Figma DS Mirror is a faithful copy of the live AngularJS app. The React platform's Tradify tokens are a modernized re-cut. They overlap in places, drift in others.

design-tokens/ (Angular reality)

  • Mirrors web-frontend-angularjs/Styles/Common/Colors.less
  • W3C token format: core / semantic / component
  • Tailwind v4 export ready
  • What Figma layers are bound to

ui-config/brand/tradify/ (current React)

  • Lives in tradify-frontend-platform/packages/ui-config
  • Tailwind v4 @theme + [data-brand="tradify"] selectors
  • Modernized palette (OKLCH naturals, gray/navy scales)
  • What components actually render against

Constraints that bound the fix

Must preserve

  • Brand toggle — every Tradify override stays inside [data-brand="tradify"]
  • Tailwind v4 @theme stays brand-agnostic
  • Access parity — never mutate the modernized shared primitives
  • Dark mode — every override has a .dark counterpart where needed

Must not

  • Pollute global @theme with Tradify-specific values
  • Mutate --color-natural-green-500 etc. (Access reads these)
  • Touch the spacing scale generation (it's regenerated by script)
  • Ship component refactors disguised as token PRs

The architectural move: override at the semantic layer (--success, --warning, --radius-card) inside the brand selector, not at the primitive layer.

E

Brand-tinted focus + button shadow

SPRINT 1 · LOW RISK

Angular buttons cast 0 1px 4px / 0.10. Default Tailwind shadow is half that. Focus rings on inputs/buttons compose ad-hoc ring-3 ring-primary/25 literals — should be one token.

[data-brand="tradify"] {
  --shadow-button:  0 1px 4px 0 rgb(0 0 0 / 0.10);
  --shadow-card-glow: 0 0 0 1px var(--color-gray-200);
  --shadow-focus-brand:
    0 0 0 3px color-mix(in oklab, var(--primary) 25%, transparent);
  --shadow-focus-danger:
    0 0 0 3px color-mix(in oklab, var(--destructive) 30%, transparent);
}

/* No @theme bridge — a self-referential
   --shadow-*: var(--shadow-*) cycles in the
   shared shadow namespace. Components consume
   bare tokens via shadow-(--token) arbitrary
   syntax, matching --shadow-xs / --input-shadow. */

Affects — Button base → shadow-(--shadow-button). Input focus-visible → shadow-(--shadow-focus-brand); invalid → --shadow-focus-danger (H · Q7). Replaces ad-hoc ring-3 ring-primary/25 literals.

D

The 30-pixel hole in the spacing scale

SPRINT 1 · TRIVIAL

Angular uses 30px frequently — card padding, section gap. Tailwind's standard scale jumps from 28 (--spacing-7) to 32 (--spacing-8). Insert one stop.

@theme {
  /* … */
  --spacing-7_5: 1.875rem;    /* 30px — Angular section gap */
  /* … */
}

Brand-agnostic — Access benefits too. One-line PR, perfect onboarding task for Nikita.

C

Semantic radius tokens

SPRINT 2 · MEDIUM

Cards in Angular are 5px. rounded-lg in React resolves to 6px. Don't retune the Tailwind scale — add semantic shape tokens. Rule: even px only, rem units. Angular's odd stops (5, 3) round to the nearest even.

Token contract · even rem

[data-brand="tradify"] {
  --radius-card:       0.375rem;  /* 6px ← Angular 5 */
  --radius-modal:      0.25rem;   /* 4px exact      */
  --radius-control:    0.25rem;   /* 4px ← Angular 3 */
  --radius-control-sm: 0.125rem;  /* 2px exact      */
  --radius-badge:      0.125rem;  /* 2px exact      */
  --radius-pill:       9999px;
}

@theme {
  --radius-card:       var(--radius-card);
  --radius-modal:      var(--radius-modal);
  --radius-control:    var(--radius-control);
  --radius-control-sm: var(--radius-control-sm);
  --radius-badge:      var(--radius-badge);
  --radius-pill:       var(--radius-pill);
}

Component swaps

  • Card → rounded-card
  • Button → rounded-control
  • Badge → rounded-badge
  • Input → rounded-control
  • Select → rounded-control

Snapshot tests catch any visual drift.

F

Type-scale fill-ins

SPRINT 2 · LOW

Angular uses 13/15/22/25/38px stops Tailwind doesn't. Don't bloat the base scale — add Tradify-scoped semantic tokens. Rule: even px only, rem units. Odd stops (13/15/25) round to nearest even; already-even stops (22/38) keep their literal value as new tokens.

[data-brand="tradify"] {
  --text-label:           0.875rem;   /* 14px ← Angular 13, aliases text-sm */
  --text-card-body:       1rem;       /* 16px ← Angular 15, aliases text-base */
  --text-card-title:      1.125rem;   /* 18px exact, aliases text-lg */
  --text-page-title:      1.25rem;    /* 20px exact, aliases text-xl */
  --text-page-emphasis:   1.375rem;   /* 22px exact, new even stop */
  --text-dashboard:       1.5rem;     /* 24px ← Angular 25, aliases text-2xl */
  --text-display-sm:      1.875rem;   /* 30px exact, aliases text-3xl */
  --text-display:         2.375rem;   /* 38px exact, new even stop */
}

@theme {
  --text-label:           var(--text-label);
  --text-card-body:       var(--text-card-body);
  --text-card-title:      var(--text-card-title);
  --text-page-title:      var(--text-page-title);
  --text-page-emphasis:   var(--text-page-emphasis);
  --text-dashboard:       var(--text-dashboard);
  --text-display-sm:      var(--text-display-sm);
  --text-display:         var(--text-display);
}

Components stop reaching for arbitrary text-[13px] literals. Storybook DS Foundations page documents the new ladder.

How the brand toggle stays intact

--color-natural-green-500
--color-orange-500

Brand-scoped via [data-brand="tradify"] selector. Where A / B / C / E / F live.

--radius-card

Layer 3 · Component

Reads Layer 2 via Tailwind utilities. Zero brand awareness in component source.
